# 🔥 Smart Calculator (AetherCore)

A modern, high-performance web calculator built with clean UI principles, advanced functionality, and production-ready frontend architecture. Designed to demonstrate strong frontend fundamentals and practical JavaScript problem-solving.

---

## 🚀 Live Features

### 🧮 Core Functionality

* Basic arithmetic operations (+, −, ×, ÷)
* Advanced operations:

  * Square (x²)
  * Square root (√)
  * Reciprocal (1/x)
  * Percentage (%)

### 🧠 Memory System

* M+ (Add to memory)
* M- (Subtract from memory)
* MR (Recall memory)
* MC (Clear memory)

### 📜 Smart History 

* Stores last 20 calculations
* Click any history item to reuse result
* Auto-updates in real-time

### ⌨️ Keyboard Support

* Full keyboard interaction
* Fast input without mouse

### 📱 Responsive Design

* Works on mobile, tablet, desktop
* No external scrolling (optimized viewport fit)
* perfect show the digits 

### 🎯 UI/UX Highlights

* Glassmorphism design
* Smooth button feedback
* Clean layout with zero clutter
* Internal scrolling only for history panel

## ⚡ Quick Start

### Option 1: Run Locally

```bash
# Clone repository
git clone https://github.com/your-username/smart-calculator.git

# Open folder
cd smart-calculator

# Open in browser
open index.html
```

### Option 2: Use Local Server

```bash
npx serve .
```
