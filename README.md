# 🔥 Smart Calculator

A perfect, modern calculator web app with:

## Features
- ✅ Basic & scientific operations
- ✅ Memory functions (M+, M-, MR, MC)
- ✅ History with persistence
- ✅ Dark/Light themes
- ✅ Sound effects
- ✅ Keyboard support
- ✅ Copy result
- ✅ Fully responsive
- ✅ PWA installable

## Quick Start
1. Open `index.html` in browser
2. Or run `npx serve .` for live reload
3. Install as PWA via browser

## Development
```
npx serve .
```
