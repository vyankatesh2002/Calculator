# 🔥 Smart Calculator

A modern web-based calculator with advanced features and PWA support.

## 🚀 Features
- Basic & scientific operations
- Memory functions (M+, M-, MR, MC)
- History with local storage
- Dark/Light mode
- Sound effects
- Keyboard support
- Copy result
- Fully responsive UI
- Installable as PWA

## ⚡ Quick Start
1. Open `index.html` in browser  
2. Or run:
   npx serve .

## 📦 Tech Stack
- HTML
- CSS
- JavaScript
